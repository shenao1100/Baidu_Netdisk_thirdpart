## 环境

- Python3.6.1 (32bit)

- Windows10

## 安装包
在cmd中 输入 pip install -r requirements.txt 即可
（requirements.txt在code文件夹中，你需要cd到code目录下才可以运行）


## 运行

下载完成所需库之后在code目录启动cmd，输入 `python main.py` 即可

---

## 功能

- 基础的文件操作 (移动，复制，删除，分享，下载...)
- 批量重命名 (没啥用)
- 成组下载
- 批量保存
- 文件取直链
- 解析分享链接
- 兼容Aria2c RPC，可以调用IDM
- 没好好做的多/单线程下载
- ...
