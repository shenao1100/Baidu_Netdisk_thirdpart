## 环境

- Python3.6.1 (32bit)

- Windows10

## 需要的python库 (一大堆)

- json
- random
- requests
- time
- os
- urllib
- threading
- subprocess
- win10toast
- base64
- hashlib
- re
- requests_toolbelt.multipart
- pyperclip
- tkinter
- ast
- qrcode
- pillow
- io

## 运行

下载完成所需库之后在根目录启动cmd，输入 `python entry.py` 即可

---

## 功能

- 基础的文件操作 (移动，复制，删除，分享，下载...)
- 批量重命名 (没啥用)
- 成组下载
- 批量保存
- 文件取直链
- 解析分享链接
- 兼容Aria2c RPC，可以调用IDM
- 没好好做的多/单线程下载
- ...
